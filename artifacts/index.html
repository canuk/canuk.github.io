Tools & Artifacts

A collection of useful HTML tools and utilities. Each tool is a standalone HTML file that can be used directly in your browser.

Available Tools

Airport Symbol Maker

Create custom aviation airport symbols with configurable runway layouts, export as SVG or PNG
EQ Feedback Trainer - Parametric Edition

Train your ears to identify EQ frequencies with interactive audio exercises and parametric equalization
Hobbs Meter Generator

Generate and download Hobbs meter SVG and PNG images for aviation hour tracking displays
SpeedRead

Speed reading app that displays text one word at a time with adjustable WPM for faster reading
Text Counter

Count words, characters, sentences, and paragraphs in your text with real-time statistics
