Tools & Artifacts

A collection of useful HTML tools and utilities. Each tool is a standalone HTML file that can be used directly in your browser.

Note: This directory is designed to host standalone HTML tools. Simply place your HTML files in the /artifacts directory and they'll be accessible at canuk.github.io/artifacts/[filename.html]

How to Add Tools

  1. Create a standalone HTML file with all necessary CSS and JavaScript inline or linked via CDN
  2. Place the file in the /artifacts directory
  3. Access it at https://canuk.github.io/artifacts/your-tool-name.html
  4. No build process or configuration needed!
