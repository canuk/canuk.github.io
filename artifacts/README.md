# Artifacts Directory

Note: This directory is designed to host standalone HTML tools. Simply place your HTML files in the /artifacts directory and they'll be accessible at canuk.github.io/artifacts/[filename.html]

How to Add Tools

Create a standalone HTML file with all necessary CSS and JavaScript inline or linked via CDN
Place the file in the /artifacts directory
Access it at https://canuk.github.io/artifacts/your-tool-name.html

No build process or configuration needed!
